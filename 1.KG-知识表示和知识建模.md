### 1.1 RDF和RDFS

**RDF ： Resource Description Framework 资源描述框架, 是数据模型**

RDF 含义:

- Resource: 页面, 图片, 视频等任何具有URI标识符的内容
- Description: 属性, 特征和资源之间的关系
- Framework: 模型, 语言和这些描述的语法

RDF 三元组模型

- 主(subject)
- 谓(predicate)
- 宾(object)

**RDF在线验证器: http://www.w3.org/RDF/Validator/**

**RDFS: RDF Schema**

RDFS 为 RDF 定义了如下词汇:

- Class
- subClassOf
- type
- Property
- subPropertyOf
- Domain
- Range
